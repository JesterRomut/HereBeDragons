> 在一个没有龙的世界当恶龙，会是什么体验？

你从纸壳中醒来，身上只有一个罗盘。
自己依稀记得，要去寻找*龙*的踪迹。
用罗盘左看又看，找了又找，
定遍一切魔法生物的居所，最终发现——

——这个世界是没有*龙*的。

混沌的大脑清醒三分，你发觉自己的身体有些不对。
啊——恶龙竟是我自己。
